OWC (Oscillating Water Column) 案例说明文档
========================================

案例概述
--------
本案例模拟了一个二维振荡水柱(OWC)波浪能转换装置的性能。
- 模型尺寸: 40m (长) × 10m (高)
- OWC装置位于中央 (x=18m 到 x=22m)
- OWC顶部有气孔开口，用于空气循环
- 在OWC前方设置四个波高检测探针

几何结构
--------
- 入口边界: 左侧 (0m)
- OWC装置: 中央 (18m - 22m)
- 出口边界: 右侧 (40m)
- 气孔开口: 顶部中央 (19m - 21m)
- 水深: 10m

边界条件
--------
- 入口: Airy波生成 (波高 0.05m, 周期 2.0s, 水位 5.0m)
- 出口: 压力出口
- OWC壁面: 无滑移壁面
- 气孔开口: 压力进出口
- 大气边界: 压力边界

监测探针
--------
四个探针位于OWC前方，用于监测波面变化:
- 探针1: (5.0, 5.0, 0.0) - 远离OWC
- 探针2: (10.0, 5.0, 0.0) - 中等距离
- 探针3: (15.0, 5.0, 0.0) - 靠近OWC
- 探针4: (17.0, 5.0, 0.0) - 最靠近OWC

运行说明
--------
1. 确保已安装OpenFOAM v2106和waveFoam v2156
2. 在案例目录下运行: blockMesh (生成网格)
3. 初始化: setFields (设置初始条件)
4. 运行: waveFoam (开始计算)
5. 或直接运行: Allrun.bat (Windows) / Allrun (Linux)

监测数据
--------
- 探针数据保存在: postProcessing/probes/
- 气室压力数据保存在: postProcessing/pressureMonitor/

后处理
--------
- 使用ParaView打开: paraFoam
- 分析探针数据: 检查alpha.water的时程变化
